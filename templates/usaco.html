USACO Tutoring

Requirements: Sign Up
  • Basic understanding of data structures
  • Knowledge of For Loops
  • Knowledge of If Statements
How does this course work?
  • Class Sessions on weekends with Skilled tutors
  • Homework on weekdays to strengthen topics covered in class sessions
  • Taught with the programming language C++
Why USACO training?

USACO is beneficial in college apps and competitive programming skills are very useful for jobs. Google, Facebook, Amazon and other major companies may ask competitive programming questions which USACO can help with. We will help guide you towards new ways of thinking and solving competitive programming questions.

This course will help you pass bronze and higher divisions. For more information on USACO please visit their website.
